    About Data Dictionary Browser

    This is an interactive Data Dictionary Browser designed for biomedical data exploration.

    Use the search bar to find specific metadata.

    Create a user to interact with the community.

    Click on the "Approve" button to approve entries and add comments, this helps the community separate good entries from less good entries.

    Keep it civil in the comment section!
